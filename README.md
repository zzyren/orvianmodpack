# Orvian Modpack for The Orvian Launcher
Extracted Prism Launcher instance containing the full Orvian modpack, prepared for use with the custom Orvian Launcher. This setup is designed specifically for a private friends-only server, ensuring easy installation, consistency, and compatibility across all players.
